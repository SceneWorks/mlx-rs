## Summary

mlx-rs is a Rust binding/wrapper for the MLX machine learning framework (Apple's array framework for Apple Silicon). The project provides Rust APIs for tensor operations, neural network layers, and model inference, along with example applications demonstrating language model inference (Mistral, Qwen3) and image classification (MNIST MLP). It also includes procedural macro utilities for generating builder patterns and macro variants for array operation functions.
