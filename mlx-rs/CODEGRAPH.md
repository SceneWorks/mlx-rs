mlx-rs is a Rust binding/wrapper library for Apple's MLX machine learning framework, providing neural network primitives, array operations, and example applications (including LLM inference with Mistral/Qwen3 and MNIST digit classification). The repository includes proc-macro utilities for generating builder patterns and variadic macros, as well as runnable examples demonstrating model loading, tokenization, and inference pipelines.
