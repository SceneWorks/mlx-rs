## Summary

mlx-rs is a Rust binding/wrapper library for Apple's MLX machine learning framework. The project provides core tensor/array operations, neural network modules, and procedural macros to ergonomically expose MLX functionality in Rust. It includes example applications demonstrating language model inference (Mistral, Qwen3) and MNIST digit classification training.
